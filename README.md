# Action Adventure Divers — St. Lucia Dive Website

Professional dive destination website for Action Adventure Divers in Soufrière, St. Lucia.

**Site:** https://1844P.github.io/aadivers-site/  
**Live URL:** https://aadivers.net

## Quick Start

1. Open **GitHub Desktop**
2. Add local repo: C:\Users\catha\Desktop\Divers\aadivers-site
3. Publish to GitHub (keep "1844P/aadivers-site" name)
4. Go to repo Settings → Pages → deploy from main branch, root folder
5. Site will be live at https://1844P.github.io/aadivers-site/

## Or via command line

`powershell
# Create the repo on GitHub first (via web), then:
git remote add origin https://github.com/1844P/aadivers-site.git
git branch -M main
git push -u origin main
`

## Then enable GitHub Pages

1. Go to https://github.com/1844P/aadivers-site/settings/pages
2. Source: Deploy from branch: main, /root
3. Save
